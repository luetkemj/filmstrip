Filmstrip.js

jQuery filmstrip carousel that supports images of varying widths and height. Fully responsive.

images courtesy of placekitten.com

github repo

The Problem

I needed a filmstrip style carousel that could support images of varying widths. This is fairly easy to solve if you are working with a static site but the carousel was required to be responsive both horizontally and vertically.

Supporting responsive width while maintaining a flush edge on the top and bottom proved problematic. Portrait images would retain a very tall vertical while landscape would become extremely short.

Example

Image of flush filmstrip

Image of same filmstrip when the viewport becomes narrower. Notice how the landscape image proportionally resizes to fit the viewport while the portrait image does not. Why would it? The width which is how this is triggered with CSS is not narrower than the viewport. Unfortunately it creates a nasty rag.

The Solution

To solve this problem I created a jQuery script that filters through all the images and reduces them proportionally to a max-width of the viewport. It then filters through again and proportionally reduces the image to the height of the shortest image.

This allows the carousel to remain flush at the top and bottom and retain a filmstrip appearance while also being fully responsive. Images should not be wider or taller than the window or viewport
